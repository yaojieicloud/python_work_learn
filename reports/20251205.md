# 📝 今日作业

## 作业需求：实现一个 **简易学生信息管理系统**

### 功能要求
1. **数据结构**
   - 使用 **列表与元组** 存储学生基本信息（姓名、学号、成绩）。
   - 使用 **字典与集合** 管理学生数据，支持快速查找与去重。

2. **字符串与正则表达式**
   - 学生学号需符合特定格式（如：`2025xxxx`），通过正则表达式进行校验。
   - 支持对学生姓名进行字符串处理（如去除空格、统一大小写）。

3. **函数**
   - 编写函数实现以下功能：
     - 添加学生信息
     - 删除学生信息
     - 查询学生信息
     - 统计平均成绩

4. **面向对象**
   - 定义一个 `Student` 类，包含属性（姓名、学号、成绩）和方法（显示信息）。
   - 定义一个 `StudentManager` 类，封装增删查改功能。

### 作业目标
- 综合运用 **基础语法、运算符、流程控制、数据结构、字符串、正则表达式、函数与面向对象**。
- 实现一个可扩展的小型系统，巩固所学知识点。
