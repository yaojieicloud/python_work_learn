# 📝 作业练习（综合第二章至第七章）

### 综合小需求：学生信息与成绩管理系统（升级版）

请编写一个 Python 程序，综合运用 **变量与数据类型、运算符与表达式、流程控制、列表与元组、字典与集合、字符串** 的知识点，实现以下功能：

1. **基本信息存储**

   - 使用元组存储学生的基本信息（姓名、年龄、身高）。
   - 使用列表存储学生的多科成绩。
2. **成绩管理**

   - 使用字典存储学生姓名与成绩对应关系。
   - 支持增删改查操作。
3. **集合去重**

   - 输入多个学生姓名，自动去除重复项。
4. **字符串处理**

   - 输入一段说明文字（如“Jie 的成绩数据”），使用字符串方法清洗（去除空格、统一大小写）。
   - 格式化输出成绩单，例如：
     ```
     学生：Jie | 年龄：25 | 身高：175.5cm
     成绩：语文85, 数学90, 英语78
     平均分：84.33 | 等级：良好
     ```
